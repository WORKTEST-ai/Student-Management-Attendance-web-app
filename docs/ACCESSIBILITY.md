# Accessibility Statement

**Last Updated: December 11, 2024**

## 1. Our Commitment

SMA (Student Management & Attendance) is committed to ensuring digital accessibility for all users, including those with disabilities.

---

## 2. Accessibility Standards

We aim to conform to **WCAG 2.1 Level AA** guidelines:

| Principle | Implementation |
|-----------|----------------|
| **Perceivable** | Alt text, color contrast, resizable text |
| **Operable** | Keyboard navigation, focus indicators |
| **Understandable** | Clear labels, consistent navigation |
| **Robust** | Semantic HTML, ARIA attributes |

---

## 3. Accessibility Features

### Visual
- ✅ High color contrast
- ✅ Resizable text (up to 200%)
- ✅ Dark/Light mode toggle
- ✅ Alt text for images
- ✅ Clear visual focus indicators

### Navigation
- ✅ Full keyboard navigation
- ✅ Skip to main content link
- ✅ Consistent navigation structure
- ✅ Descriptive link text

### Forms
- ✅ Form field labels
- ✅ Error messages linked to fields
- ✅ Required field indicators
- ✅ Form validation messages

### Screen Readers
- ✅ Semantic HTML structure
- ✅ ARIA labels where needed
- ✅ Heading hierarchy
- ✅ Landmark regions

---

## 4. Known Limitations

We are actively working to address:
- Some charts may need improved descriptions
- PDF exports may need accessibility review
- Some older browser compatibility
