# **App Name**: SMA

## Core Features:

- User Authentication and Roles: Secure authentication system with roles for Admin, Teacher, and Student, managing access to different features and data.
- Admin Panel: Comprehensive dashboard for managing users, classes, sections, subjects, and attendance records. Includes bulk import via CSV and export functions.
- Teacher Attendance Recording: Teachers can record and filter attendance by class, section, subject, and date. Includes 'Mark All Present', save draft and finalize functionalities.
- Student Attendance Overview: Student panel provides an overview of attendance percentages, latest activity, and the ability to download personal reports.
- Automated Report Generation: Automatically generate insights on student attendance based on historical data, flagging patterns that a human tool user can act upon.
- Data Export: Export attendance reports in CSV and PDF formats for various classes and date ranges.
- Audit Log: Track and log all admin actions, ensuring accountability and transparency within the system.

## Style Guidelines:

- Primary color: Calm blue (#5DADE2) to evoke trust and stability in an educational environment.
- Background color: Light gray (#F0F4F8) for a clean, professional look that minimizes distraction.
- Accent color: Green (#A3E4D7) for CTAs, highlighting positive actions like 'Mark Present'.
- Body and headline font: 'Inter', a versatile sans-serif for clear readability across the interface.
- Use a consistent set of educational icons (students, books, classrooms) that follow a flat design style.
- Responsive layout with a left sidebar (collapsible) for navigation and a clean content grid for data display.
- Subtle transitions and animations on form submissions and data updates to enhance user experience without being distracting.
